VIM配置

一、简单设置

set relativenumber    " 相对行号
set cursorline         " 突出显示当前行
set ruler             " 打开状态栏标尺
set shiftwidth=4     " 设定 << 和 >> 命令移动时的宽度为 4
set softtabstop=4     " 使得按退格键时可以一次删掉 4 个空格
set tabstop=4         " 设定 tab 长度为 4
set nobackup        " 覆盖文件时不备份

二、设置配色

  1. checkout主题

    git clone git://github.com/altercation/solarized.git

  2. 复制solarized.vim到.vim/colors目录

    cp solarized/vim-colors-solarized/colors/solarized.vim ~/.vim/colors/

  3. 更改~/.vimrc

    syntax enable

if has('gui_running')
   set background=light
else
   set background=dark
endif

colorscheme solarized

三、安装中文help

  1. 下载中文帮助的文件包:

    $wget http://nchc.dl.sourceforge.net/sourceforge/vimcdoc/vimcdoc-1.5.0.tar.gz

  2. 进入文件夹安装：

    $sudo ./vimcdoc.sh -i

  3. 设置help为中文

    set helplang=

四、使用ctags

  1. 安装

    sudo apt-get install ctags

  2. 生成tags

    ctags -R

  3. 在vim中开启tags

    :set tags=./tags

  4. 使用

    C-] 跳到定义
C-T 返回

  5. 帮助

    :help usr_29
