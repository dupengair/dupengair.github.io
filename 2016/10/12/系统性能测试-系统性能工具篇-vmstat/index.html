系统性能工具篇(vmstat)

可用选项

delay: 更新间隔秒数，不指定时为启动到目前的平均值
count: 更新次数，指定delay时默认无限
    -a: 显示活跃和非活跃内存
    -f: 启动到现在的fork数量，包含fork、vfork、clone
    -m: SLAB模式 
    -s: 显示内核事件统计信息
    -d: 磁盘模式
    -D: 磁盘活动统计数据
    -p: 分区模式

虚拟内存模式下（默认）：

# vmstat 1 2
    procs -----------memory---------- ---swap-- -----io---- -system-- ----cpu----
     r  b   swpd   free   buff  cache   si   so    bi    bo   in   cs us sy id wa
     2  0      0 421892      0 115372    0    0     0     0  867 1058  5  7 88  0
     0  0      0 422112      0 115376    0    0     0     0 1320 1732  3  2 95  0

    第一行产生的报告时上次重启后的平均值，应该排除

    Procs:
            r 正在运行或等待cpu调度的经常数量
            b 不可中断睡眠态的进程数量
    memory:
            swpd    虚拟内存
            free    空闲内存 
            buff    缓冲区
            cache    缓存
            inact    非活跃内存（-a）
            active    活跃内存（-a）
    swap:
            si 磁盘交换入内存数量
            so 内存交换到磁盘数量
    io:
            bi 从块设备接收到的块
            bo 发送到块设备的块
    system:
            in: 每秒中断数，含时钟
            cs: 每秒上下文切换数
    cpu:
            us: 用户态运行时间，包含nice
            sy: 内核态运行时间
            id: 空闲时间
            wa: io等待时间
            st: 虚拟机时间，未知

        merged: 合并后分组的写（导致一个IP）
        sectors: 成功写入的扇区
        ms:    写入花费的毫秒
    IO:
        cur: 正在进行的IO
        sec: IO花费的秒数
