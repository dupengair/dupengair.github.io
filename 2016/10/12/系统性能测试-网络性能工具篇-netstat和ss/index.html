  1. Active Internet connections

    Proto: socket使用的协议（tcp、udp、raw、tcp6、udp6、raw6）
Recv-Q
    Established: socket所在的应用程序还未读取的字节数
    Listening: 包含当前syc_backlog的最大值 ？
Send-Q
    Established: 已发出还未被远程主机确认的字节数
    Listening: 包含当前syc_backlog的最大值 ？
Local Address: 本地IP、端口
Foreign Address: 远程主机IP、端口
State: tcp才有状态
    ESTABLISHED: 已建立连接
    SYN_SENT : 主动打开
    SYN_RECV : 被动打开
    FIN_WAIT1: 主动关闭
    FIN_WAIT2: 等对方关闭
    TIME_WAIT: 等待2MSL
    CLOSE    : 已关闭
    CLOSE_WAIT: 被动关闭，对方已关闭，等待本地关闭
    LAST_ACK : 本地已执行关闭，等对方确认
    LISTEN   : 正在监听
    CLOSING  : 两端已执行关闭，仍然有数据没有发送完成
    UNKNOWN  : 未知状态

  2. Active UNIX domain sockets

    Proto : socket使用的协议，通常是unix
RefCnt: 应用计数，附加到socket的进程数
Flags : 
    SO_ACCEPTION (ACC): 用于还未连接的socket，进程正在accept等待请求
    SO_WAITDATA (W)
    SO_NOSPACE (N)
Type  : socket的访问类型
    SOCK_DGRAM : 无连接模式，UDP
    SOCK_STREAM: 流式套接字，TCP
    SOCK_RAW   : 原始套接字
    SOCK_RDM   : 提供可靠消息传递
    SOCK_SEQPACKET: 连续数据包的socket ？
    UNKNOWN    : 面向未来的类型
Stat:
    FREE      : socket还未被分配
    LISTENING : 监听链接请求
    CONNECTING: 正在建立连接的socket
    CONNECTED : 已经连接的socket
    DISCONNECTING: 已经断开连接的
    (empty)      : 没有连接到任何socket
    UNKNOWN   : 永远不会发生
I-Node: socket的inode编号
Path  : 附加到socket的应用程序名称

二、ss（超有用）

  1. 参数

    -alnp: 同netstat
-e: 详细socket信息
-m: socket内存信息
-i: TCP内部信息
-o: 定时器信息
-s: summary
-t: tcp
-u: udp
-w: raw
-x: unix
-4/6

三、iptraf

  1. 功能项
    • IP traffic monitor
      • 统计TCP/UDP连接流量
    • General interface statistics
      • 统计IP包流量
    • Detailed interface statistics
      • 根据协议统计流量
    • Statistaical breakdowns
      • 根据包大小分布或TCP/UDP端口统计流量
    • LAN station monitor
      • 根据MAC地址统计流量
