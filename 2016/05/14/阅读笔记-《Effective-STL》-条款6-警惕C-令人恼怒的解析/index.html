#一、产生

  1. 代码

    /* 传一对istream_iterator给list的区间构造函数，把int从文件拷贝到list中 */
ifstream dataFile("ints.dat");
list<int> data(istream_iterator<int>(dataFile), istream_iterator<int>());
    • 实际在运行时，什么都没做。不会从文件中读出任何数据。它甚至不会建立一个list。因为第二句并不声明list，而且它也不调用构造函数，而是一个函数声明

#二、分析

  1. 函数声明

    int f(double (d));
int f(double d);
int f(double);
int g(double (*pf)());
int g(double pf());
int g(double ());
    • 1~3等价，参数名左右的括号，以及参数名本身可以忽略
    • 4~6等价，参数为函数指针，作为参数的函数名可以忽略
    • 函数声明只关心参数类型，参数名可以忽略，单独的括号声明了存在指向函数的指针的参数

  2. 错误代码

    list<int> data(istream_iterator<int>(dataFile), istream_iterator<int>());
    • 声明函数data，返回类型list，带有两个参数：
      • 参数名为dataFile。类型istream_iterator<int>，dataFile被忽略。
      • 第二个参数没有名字。类型是指向没有参数而且返回istream_iterator<int>的函数的指针。

  3. 常见错误

    class Widget; 
Widget w();        // 不是构造对象w
    • 声明函数w，没有参数且返回Widget类型对象

#三、修改

  1. 加括号，可能编译器不识别

    list<int> data((istream_iterator<int>(dataFile)), istream_iterator<int>());

  2. 通行做法，分开声明

    ifstream dataFile("ints.dat");
istream_iterator<int> dataBegin(dataFile);
istream_iterator<int> dataEnd;
list<int> data(dataBegin, dataEnd);
