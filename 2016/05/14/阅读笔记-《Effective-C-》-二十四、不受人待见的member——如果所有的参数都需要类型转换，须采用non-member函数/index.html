title: 二十四、不受人待见的member——如果所有的参数都需要类型转换，须采用non-member函数
date: 2016-05-13 17:02:03
categories: [阅读笔记, 《Effective C++》]
tag: 《Effective C++》
toc: true

member函数确实不受待见，又来一条鄙视它的法制——如果所有参数都需类型转换，请采用non-member函数。

#一、看一个实例

  1. 让class支持隐式类型转换通常不是好主意，不过常见的例外是建立数值类型时。

    如一个有理数类：
class Rational {
public:
    Rational(int numerator = 0, int denominator = 1);
    int numerator() const;
    int demoninator() const;
private:
    …
};

对有理数实现operator*操作，当它为member函数时：
class Rational {
public:        
    …
    const Rational operator* (const Rational& rhs) const;
};

#二、问题

  • 实际应用中的有理数乘法：

    Rational oneEighth(1, 8);
Rational oneHalf(1, 2);
Rational result = oneEighth * oneHalf;        // OK!
result = oneHalf * 2;                // OK! 隐式转换
result = 2 * oneHalf;                // Bomb!

  • 不满足乘法交换律，分析一下原因，将上面的表达式写成函数形式：

Rational result = oneEighth.operator*(oneHalf);        // OK!
result = oneHalf.operator*(2);                // OK! 隐式转换
result = 2.operator*(oneHalf);                // Bomb!
  • 很明显，整数2不存在class，也就没有operator*()成员函数。

  • 对于operator*()成员函数来说，2为左操作数，因此2并不在其参数列表中，也就无法进行隐式类型转换。虽然此时编译器会在global作用域内寻找这样形式的non-member函数：

    const Rational operator* (const Rational& rhs, const Rational& rhs);

  • 很可惜没有找到，所以报错。而对于第二个表达式，由于整数2经过隐式转换已经成了Rational对象，其运算过程可以解析如下：

    const Rational temp(2);
result = oneHalf.operator*(temp);

  • 而如果把class的构造函数声明为explicit，不允许进行隐式类型转换，那么上面后两个整数参与计算的表达式都无法通过编译。

#三、结论

  • 以上的事实表明，之所以无法进行类型转换，是因为参数不在参数列表中。

#四、解决办法

  • 要解决这个问题，很明显要让operator*()成为一个non-member函数，这样所有的参数才会在参数列表中，能够进行类型转换：

    class Rational {
public:        
    …
};

const Rational operator* (const Rational& rhs, const Rational& rhs)
{
    return Rational( lhs.numerator() * rhs.numerator(),
             lhs.denominator() * rhs.denominator() );
}

result = oneHalf * 2;                // OK! 隐式转换
result = 2 * oneHalf;                // OK! 隐式转换

#五、观察

  1. member函数的反面是non-member函数，不是friend函数。假设“如果一个与某class相关的函数不该成为一个member，就该是个friend”是不成立的。
  2. 无论何时，可以避免friend函数就应该避免
  3. 不能够因为一个函数不成为member，就自动让它成为friend

#六、结论
如果需要为某个函数的所有参数进行类型转换，这个函数必须是个non-member。
