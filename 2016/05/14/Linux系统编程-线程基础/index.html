title: 线程基础
date: 2016-05-13 17:01:55
categories: Linux系统编程
tag: Linux系统编程
toc: true

  1. 线程标识

    • pthread_t: 非负整数
    • 可移植的做法int pthread_equal(pthread_tid1, pthread_tid2)
    • 获取自身pthread_t pthread_self(void)

  2. 线程创建

    • 不能保证运行顺序
    • 可访问进程空间
    • 继承父线程的浮点环境和信号屏蔽字，清空未决信号
    • linux使用clone系统调用实现pthread_create，使子进程可以共享父进程的一定数量的环境

  3. 线程退出

    • 三种方式：
      • 返回
      • 调用pthread_exit
      • 被其它线程取消

    • 向其它线程传递退出信息：

      void pthread_exit(void *)
      • 其它线程调用int pthread_join(pthread_t, void**)获取到这个指针
      • 当线程厨房分离状态时，pthread_join调用返回EINVAL

    • 线程清理

      void pthread_cleanup_push(void (*func)(void*), void*)
void pthread_cleanup_pop(int)

  4. 线程同步

    • 原子操作
      • 只需要一个指令周期完成的存储操作
    • 互斥量
      • 死锁：自身加锁、两个线程以相反的顺序加锁
    • 读写锁
      • 读模式加锁、写模式加锁、无锁

  5. 条件变量

    • 线程集合点
    • 等待线程先锁再wait，wait函数保证返回时是锁住状态，避免接下来基于锁对条件进行的操作出现同步问题
    • 注意一定要条件改变后再发送信号
    • 对条件的操作：改变->判断 都要锁保护
    • 书中未提，先发送了信号再等待，会出现什么问题 -> 信号丢失
