#一、继承与组合

  • public 继承：is-a
  • 组合：has-a

#二、组合

  1. 组合发生在应用域的对象之间（生活中的实体对象），表现出has-a
  2. 组合发生于实现域内（程序中的功能实体，如缓冲区），表现出is-implemented-in-terms-of

#三、实例：用STL的list实现一个自己的set

  1. 继承

    template<typename T>
class Set: public std::list<T> {…};
    • 这是错误的做法，因为set中不能有重复元素，从面向对象的意义上来说，set不是一个list，不存在is-a的关系，所以不应该用继承

  2. 组合

    template<typename T>
class Set {
public:
    bool member(const T& item) const;
    void insert(const T& item);
    void remove(const T& item);
    std::size_t size() const;
private:
    std::list<T> rep;
};

template<typename T>
bool Set::member(const T& item) const
{
    return std::find(rep.begin(), rep.end(), item) != rep.end();
}

template<typename T>
void Set::insert(const T& item) 
{
    if(!member(item)) return push_back(item);
}

template<typename T>
void Set::remove(const T& item) 
{
    typename std::list<T>::iterator it=
        std::find(rep.begin(), rep.end(), item);
    if(it != rep.end()) rep.erase(it);
}

template<typename T>
std::size_t Set::size() const
{
    return rep.size();
}
