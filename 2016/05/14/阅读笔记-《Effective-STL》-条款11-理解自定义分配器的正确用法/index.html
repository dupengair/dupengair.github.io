#一、自定义分配器，将容器元素放在堆上

  1. 管理共享内存的堆

    void* mallocShared(size_t bytesNeeded);
void freeShared(void *ptr);

  2. 自定义分配器

    template<typename T>
class SharedMemoryANocator {
    public:
        ...
        pointer allocate(size_type numObiects, const void *localityHint = 0)
        {
            return static_cast<pointer>(mallocShared(numObiects * sizeof(T)));
        }
        void deallocate(pointer ptrToMemory, size_ type numObjects)
        {
            freeShared(ptrToMiemory);
        }
...
};

  3. 使用分配器

    • 期望在共享内存上建立vector对象

      typedef vector<double, SharedMemoryAllocator<double> > SharedDoubleVec;
...
{
    SharedDoubleVec v;  // 企图建立一个元素在共享内存中的vector
    ...                 // 结束这个块
}
      • 希望容器的每个元素都建立在共享内存上(通过自定义分配器)，但容器对象本身建立在了栈上，对于运行时系统，不会放到共享内存，因此并没有达到将容器放到共享内存的目的
* 因此，需要将容器本身建立在共享内存才行，需要用到placement new

        void *pVectorMemory = mallocShared(sizeof(SharedDoubleVec)); // 分配足够的共享内存来容纳一个SharedDoubleVec对象
        SharedDoubleVec *pv = new (pVectorMemory) SharedDoubleVec;   // 使用placement new来在那块内存中建立一个SharedDoubleVec对象
        ...
        pv->~SharedDoubleVec();    // 销毁共享内存中的对象
        freeShared(pVectorMemory); // 销毁原来的共享内存块

    + 代码忽略了mallocShared可能返回一个null指针
    + 通过placement new调用容器自身的构造函数，才能调用自定义分配器把容器放到共享内存

#二、自定义分配器，将不同对象放置到不同的堆空间

  1. 两个不同的堆管理类

    class Heap1 {
    public:
        ...
        static void* alloc(size_t numBytes, const void *memoryBlockToBeNear);
        static void dealloc(void *ptr);
        ...
};
class Heap2 { ... };     // 有相同的alloc/dealloc接口
    • 接口相同，只是不同的堆空间

  2. 自定义分配器，可以将不同的对象通过分配器参数指定分配到不同的堆空间中

    template<typenameT, typename Heap>
class SpecificHeapAllocator {
    public:
        pointer allocate(size_type numObjects, const void *localityHint = 0)
        {
            return static_cast<pointer>(Heap::alloc(numObjects * sizeof(T), localityHint));
        }
        void deallocate(pointer ptrToMemory, size_type numObjects)
        {
            Heap::dealloc(ptrToMemory);
        }
        ...
};

  3. 应用时，建立不同的容器对象指定不同的堆管理类

    vector<int, SpecificHeapAllocator<int, Heap1> > v; // 把v和s的元素放进Heap1
set<int, SpecificHeapAllocator<int Heap1> > s; 

list<Widget, SpecificHeapAllocator<Widget, Heap2> > L; // 把L和m的元素放进Heap2
map<int, string, less<int>, SpecificHeapAllocator<pair<const int, string>, Heap2> > m;
    • 注意，重要的一点是Heap1和Heap2是类型而不是对象。如果Heap1和Heap2是对象而不是类型，那么它们将是不等价的分配器，那就违反了分配器的等价约束.
