文章目录

1、安装VIM
sudo apt-get install vim

2、安装经典菜单指示器
sudo add-apt-repository ppa:diesch/testing
sudo apt-get update
sudo apt-get install classicmenu-indicator

3.安装系统指示器SysPeek
sudo add-apt-repository ppa:nilarimogard/webupd8
sudo apt-get update
sudo apt-get install syspeek

4、安装openssh-server
sudo apt-get install openssh-server

5、安装samba
sudo apt-get install samba

  1. 配置samba
    1) 新建共享目录
    sudo mkdir /home/dupengair/Development
    sudo chmod 777 /home/dupengair/Development

2）配置samba
sudo vim /etc/samba/smb.conf

打开”security = user”

编辑共享目录：
[shared]
comment = almond’s
path = /home/shared
browseable = yes
writable = yes
valid users = almond
read only = no

3) 添加samba用户
sudo useradd almond
sudo smbpasswd -a almond

  1. 设置时区和ntp校时
    sudo tzselect
    sudo ntpdate cn.pool.ntp.org

8、添加新用户
sudo adduser almond
添加到sudoer：
sudo visudo
root ALL=(ALL) ALL下面添加一行：
almond ALL=(ALL) ALL

  1. 设置ssh证书登陆
    server：
    1) 制作证书
    切换到almond用户
    ssh-keygen -t rsa
    cd /home/ssh1/.ssh
    mv id_rsa.pub authorized_keys
    [重要!]保证权限（默认即可，否则按如下调整）：
    authorized_keys –> 600
    .ssh –> 700
    Home Directory –> 711
    Keys (id_dsa,id_rsa) –> 600
    Public Keys (id_dsa.pub,id_rsa.pub) –> 644
    2）配置sshd
    vi /etc/ssh/sshd_config #编辑
    PermitRootLogin no #禁用root登录
    PasswordAuthentication no #禁用账号密码登录
    RSAAuthentication yes #RSA认证
    PubkeyAuthentication yes #公匙认证
    AuthorizedKeysFile %h/.ssh/authorized_keys #公匙路径
    3) 重启SSH服务
    /etc/init.d/ssh restart

3) 配置putty
puttygen:Conversions->Import key->save private key 根据RSA证书生成ppk，选私钥id_rsa
putty：Connection-SSH-Auth -> 选择生成的ppk证书
Connection-Data,Auto-login username -> ssh1
切换到Session, save session

  1. Mac端配置ssh登陆
    Mac端生成公钥，添加到linux公钥中：authorized_keys
