第二章 构造函数语意学

一、合成default constructor

  1. class 带有default constructor 的 member
    • 调用member的default constructor
  2. class 继承自带有default constructor 的 base class
    • 调用base class的default constructor
  3. class 带有virtual function
    • 初始化vptr指向vtbl
  4. class 带有virtual base class
    • 初始化bptr指向virtual base class, 普通类在使用同一个基类的成员时要指明作用域（否则有二义性），所以不存在问题

二、合成default copy constructor

  1. class 带有default copy constructor 的 member
  2. class 继承自带有default copy constructor 的 base class
    • 1、2同上
  3. class 带有virtual function
    • 重设vptr，当Derived d; Base b=d;时，调整d的vptr指向Base，此时对象slicing
  4. class 带有virtual base class
    • 重设bptr，当Derived d; Base b=d;时，b的bptr指向d的Base offset

三、程序转化

void foo(X x0);

X xx;
foo(xx);

转化为：
X __temp0;
__temp0.X:X(xx);
foo(__temp0);

void foo(X& x0);
  • 即形参值传递变为临时对象+引用传递

四、返回值初始化

  1. 定义

    X bar() {
    X xx;
    …
    return xx;
}    

转化为：
void bar(X& __result) {
    X xx;
    xx.X::X();

    __result.X::X(xx);
    return;
}
    • 即返回值变为临时变量+引用参数

  2. 调用

    X xx = bar();

转换为：
X xx;
bar(xx);

  3. 返回值优化

    X bar() {
    return X();
}
转化为：
void bar(X& __result) {        
    __result.X::X();
    return;
}
    • 即不再产生临时对象，直接对参数调用构造函数

  4. 不要随意使用memset和memcpy，可能会破坏编译器内部产生的member

五、初始化

  1. 使用初始化list

    • 初始化 reference member
    • 初始化 const member
    • 调用base class的constructor
    • 调用member class的constructor

  2. list的初始化顺序由class member的声明顺序决定，不由list顺序决定

  3. list在构造函数体之前执行
