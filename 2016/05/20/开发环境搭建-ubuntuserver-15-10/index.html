1. ssh

安装：
sudo apt-get install openssh-server
确认：
ps -ef | grep ssh: /usr/sbin/sshd -D

2. 添加新用户almond（用于ssh、samba等）

sudo adduser almond
添加到sudoer：
sudo visudo
root ALL=(ALL:ALL) ALL下面添加一行：
almond ALL=(ALL:ALL) ALL

3. 安装vim

sudo apt-get install vim

4. 配置samba

sudo apt-get update
sudo apt-get install samba samba-common
sudo mkdir /home/public
sudo chmod 777 /home/share
sudo vim /etc/samba/smb.conf

把almond用户添加到samba用户：
sudo smbpasswd -a smbuser
sudo service smbd restart

5. 设置时区和ntp校时

sudo tzselect
sudo cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
sudo apt-get install ntpdate
sudo ntpdate cn.pool.ntp.org

6. 设置ssh证书登陆

server：
1)给almond用户制作证书
ssh-keygen -t rsa
cd /home/almond/.ssh
cp id_rsa.pub authorized_keys
[重要!]保证权限：
authorized_keys –> 600
.ssh –> 700
Home Directory –> 711
Keys (id_dsa,id_rsa) –> 600
Public Keys (id_dsa.pub,id_rsa.pub) –> 644
2）配置sshd
vi /etc/ssh/sshd_config #编辑
PermitRootLogin no #禁用root登录
PasswordAuthentication no #禁用账号密码登录
RSAAuthentication yes #RSA认证
PubkeyAuthentication yes #公匙认证
AuthorizedKeysFile %h/.ssh/authorized_keys #公匙路径

sudo service sshd restart #重启SSH服务

3) 给putty制作证书
puttygen:Conversions->Generate->save private key 保存秘钥putty_rsa(注意不要关窗口)
把窗口上显示的公钥复制到文件putty_rsa.pub，然后拷贝到samba目录 /home/public/下

server：将putty生成的公钥添加到authorized_keys:
cat /home/public/putty_rsa.pub >> .ssh/authorized_keys
putty：Connection-SSH-Auth -> 选择秘钥
Connection-Data,Auto-login username ->almond 填登录名
切换到Session, save session
然后输入IP即可用秘钥登录

7. 安装中文语言包

sudo apt-get install language-pack-zh-hans (简体)
sudo apt-get install language-pack-zh-hant（繁体）

8. 安装tofrodos工具

sudo apt-get install tofrodos

9. 安装curl

sudo apt-get install curl
