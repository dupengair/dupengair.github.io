安装docker（默认版本1.6，不推荐）

sudo apt-get install docker.io

升级docker或安装最新版本（1.10，推荐）

1. 更新系统

sudo apt-get update

2. 导入证书

sudo apt-get install apt-transport-https ca-certificates

3. 获取docker仓库公钥

sudo apt-key adv --keyserver hkp://p80.pool.sks-keyservers.net:80 --recv-keys 58118E89F3A912897C070ADBF76221572C52609D

4. 创建文件，添加apt源

sudo sh -c "echo deb https://apt.dockerproject.org/repo ubuntu-wily main > /etc/apt/sources.list.d/docker.list"

5. 更新ubuntu

sudo apt-get update

6. 清理旧仓库（此步骤测试可不需）

sudo apt-get purge lxc-docker

7. 查看apt源是否更新成功（有最新版本docker-engine）

sudo apt-cache policy docker-engine

8. 安装最新docker-engine，会提示旧版本docker.io将被删除

sudo apt-get install docker-engine

此步骤需翻墙，参考《ubunt sever1510下使用openvpn连接switchvpn》
