三、make app

1、common/license/inc/encryption.h缺失，屏蔽common/makefile Line 8不编译license，可以编译通过common里其它模块
2、Framework/helpers/processhelper.cpp 添加#include <sys/stat.h>
3、Framework/helpers/CryptHelper.cpp、CryptHelper.h 缺失，
   Framework/core/Users.cpp缺失，Framework无法编译通过，
   先编译一遍app生成需要的库，出错停止后在主Makefile中注释掉Framework再编译其它的部分。
   Line 148: @$(MAKE) --no-print-directory -C Framework
4、VideoSphere/Library/Ambarella_A5/AHD.cpp Line 319 把参数里的Framework::Helper::Guid::Guid::Guid 改为：Framework::Helper::Guid::Guid
5、VideoSphere/Makefile Line 27：注释掉bankcamera模块，里面会用到license模块
6、VideoSphere/library/下的HostManager、StorageDeviceHandler、RtspServer、JpegStreamer、VSClientInterface这几个模块会用到Framework/helpers/CryptHelper模块，无法编译通过，其它的没问题，注释掉Line 38 39 43 44 46

2、Framework/helpers/CryptHelper.cpp、CryptHelper.h 缺失
