一、使用accumulate

  1. 现有的方式

    • count统计区间中有多少等于某个值的元素
    • count_if统计有多少元素满足一个判断式
    • min_element和max_element统计区间中的最小和最大值

  2. accumulate

    • 对数列进行运算

      T accumulate(beg, end, T initVal)
T accumulate(beg, end, T initVal, op)
      • 形式一计算initVal与每个元素值的总合：initVal+=elem，形式二计算initVal与每个元素op的结果：initVal=op(initVal, elem)
      • 如果数列为空（beg == end）两者都返回initVal
      • op不能改动实参
      • 复杂度：线性

    • 计算区间的和

      list<double> ld;
...
double sum = accumulate(ld.begin(), Id.end(), 0.0); // 计算它们的和，从0.0开始

    • 注意initVal类型要和元素类型一致

      double sum = accumulate(ld.begin(), Id.end(), 0); // 计算它们的和，从0开始；这不正确
      • 如果初始值是int 0，所以accumulate内部就会使用一个int来保存它计算的值。那个int最后变成accumulate的返回值，而且它用来初始化和变量。这代码可以编译和运行，但和的值可能不对。不是保存真的double的list的和，它可能保存了所有的double加起来的结果，但每次加法后把结果转换为一个int

二、计算区间

  1. 计算容器中的字符串的长度和

    • 必须知道和的开始。在我们的例子中，它是0

    • 必须知道每次看到一个新的字符串时怎么更新这个和

      string::size_type stringLengthSum(string::size_type sumSoFar, const string& s)
{
    return sumSoFar + s.size();
}

set<string> ss; 
... 
// 把lengthSum设为对ss中的每个元素调用stringLengthSum
string::size_type lengthSum = accumulate(ss.begin(), ss.end(), 0, stringLengthSum);

  2. 计算数值区间

    • 可以使用标准multiplies仿函数类

      vector<float> vf; 
... 
// 把product设为对vf中的每个元素调用multiplies<float>的结果，用1.0f作为初始统计值
float product = accumulate(vf.begin(), vf.end(), 1.0f, multiplies<float>());

  3. 计算point的区间的平均值

    struct Point {
    Point(double initX, double initY): x(initX), y(initY) {}
    double x, y;
};

list<Point> lp;
...
// 对Ip中的point求平均值
Point avg = accumulate(lp.begin(), lp.end(), Point(0, 0), PointAverage());

    • 求平均值，有副作用的实现

      class PointAverage: public binary_function<Point, Point, Point> { // 参见条款40
    public:
        PointAverage(): numPoints(0), xSum(0), ySum(0) {}
        const Point operator()(const Point& avgSoFar, const Point& p) {
            ++numPoints;
            xSum += p.x;
            ySum += p.y;
            return Point(xSum/numPoints, ySum/numPoints);
        }
    private:
        size_t numPoints;
        double xSum;
        double ySum;
};
      • 因为带有状态，且在仿函数内部被修改，成员变量numPoints、xSum和ySum的修改造成了副作用，因为传给accumulate的仿函数PointAverage带有两个参数。这个结果可能未定义

三、accumulate与for_each

  1. 不同点

    • 首先，accumulate的名字表示它是一个产生区间统计的算法，for_each只是对区间的每个元素进行一些操作
    • 其次，accumulate直接返回统计值，而for_each返回一个函数对象，必须从这个对象中提取想要的统计信息。在C++里，那意味着必须给仿函数类添加一个成员函数来获得统计信息

  2. 使用for_each实现计算point的区间的平均值

    struct Point {...); // 同上
class PointAverage: public unary_function<Point, void> { 
    public:
        PointAverage(): xSum(0), ySum(0), numPoints(0) {}
        void operator()(const Point& p)
        {
            ++numPoints;
            xSum += p.x;
            ySum += p.y;
        }
        Point result() const
        {
            return Point(xSum/numPoints, ySum/numPoints);
        }
    private:
        size_t numPoints;
        double xSum;
        double ySum;
};

list<Point> Ip;
...
Point avg = for_each(lp.begin(), lp.end(), PointAverage()).result；
