基础算法学习(十七) —— 归并排序

一、概念

  1. 通常涉及完全二叉树的排序算法，效率都不会低

  2. 思路

    • 将含有n个记录的序列看作n个子序列，进行两两归并，然后向上合并，继续两两归并，直到有序，称为2路归并

      5  8 7  2 4  1 3  6
 \/   \/   \/   \/
 58   27   14   36
   \ /       \ /
   2578      1346
      \    /
     12345678

二、递归实现

  1. 2路归并过程

    • 不断细分，直到左右元素相等，再一一返回，两两合并

      void MergeSort(TList *l)
{
    MSort(l->r, l->r, 1, l->length);
}

void MSort(int SR[], int TR1[], int s, int t)
{
    int m;
    int TR2[MAXSIZE+1];

    if(s == t)        // 细分到记录为1
        TR1[s] = SR[s];
    else
    {
        m = (s+t)/2;
        // 将SR归并为有序的TR2
        MSort(SR, TR2, s, m);
        MSort(SR, TR2, m+1, t);
        Merge(TR2, TR1, s, m, t);
    }
}

  1. 比较与合并

    • 将记录左半与右半元素一一比较，放入TR数组，完成合并

      void Merge(int SR[], int TR[], int i, int m, int n)
{
    // 左边：i~m, 右边：j~n
    int j, k, l;
    // 从左右中挑选小的入列
    for(j=m+1, k=i; i<=m&&j<=n; k++)
    {
        if(SR[i] < SR[j])
            TR[k] = SR[i++];
        else
            TR[k] = SR[j++];
    }

    // 左边还有剩余，直接入列
    if(i <= m)
        for(l=0; l<=m-i; l++)
            TR[k+l] = SR[i+l];

    // 右边还有剩余，直接入列
    if(j <= n)
        for(l=0; l<=n-j; l++)
            TR[k+l] = SR[j+l];
}

三、非递归实现

  1. 两两细分

    • 从最小序列记录个数为1开始，比较自然

      void MergeSort2(TList *l)
{    
    int *TR = (int *)malloc(l->length*sizeof(int));
    int k = 1;

    // 间距k从1开始翻倍，控制归并的粒度(1x1, 2x2, 4x4, ...)
    while(k < l->length)
    {
        // 将无序数组两两归并到TR
        MergePass(l->r, TR, k, l->length);
        k = 2*k;
        // 将两两有序的数组归并回l->r
        MergePass(TR, l->r, k, l->length);
        k = 2*k;
    }

    free(TR);
}

  2. 合并排序

    void MergePass(int SR[], int TR[], int s, int n)
{
    int i=1;
    int j;

    while(i <= n-2*s+1)
    {
        Merge(SR, TR, i, i+s-1, i+2*s-1);
        i = i+2*s;
    }

    if(i < n-s+1)
        Merge(SR, TR, i, i+s-1, n);
    else
        for(j=i; j<=n; j++)
            TR[j] = SR[j];
}

四、输出

int a[MAXSIZE] = { 50, 10, 90, 30, 70, 40, 80, 60, 20 };

TList l;

InitList(&l, a);
PrintList(&l, "original list: ");
MergeSort(&l);
PrintList(&l, "merge sorted list: ");

InitList(&l, a);
PrintList(&l, "original list: ");
MergeSort2(&l);
PrintList(&l, "merge sorted list2: ");

五、特点

  • 复杂度O(nlogn)
  • 排序稳定，不存在跳跃
  • 需要将原始记录完全放入内存进行比较，递归实现下栈深度大，因此内存占用高，空间上的性能损耗大
  • 高效且稳定
  • 应尽量考虑非递归的方法
